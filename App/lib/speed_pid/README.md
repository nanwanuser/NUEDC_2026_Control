# 电机速度环 PID

## 离散数学公式

速度误差：

```text
e(k) = r(k) - y(k)
```

其中 `r(k)` 是目标输出轴转速，`y(k)` 是编码器测得的输出轴转速。

比例项：

```text
P(k) = Kp * e(k)
```

积分项：

```text
I(k) = clamp(I(k-1) + Ki * Ts * e(k), Imin, Imax)
```

微分项采用“对测量值微分”，避免目标转速阶跃时出现微分冲击：

```text
D_raw(k) = -(y(k) - y(k-1)) / Ts
D(k) = D(k-1) + alpha * (D_raw(k) - D(k-1))
alpha = 2*pi*fc*Ts / (1 + 2*pi*fc*Ts)
```

最终输出：

```text
u(k) = clamp(Kp*e(k) + I(k) + Kd*D(k), Umin, Umax)
```

控制器带积分限幅和条件积分抗饱和。建议输出单位使用占空比百分数 `0~100`。

## 阶跃整定建议

1. 先保持 PID 不参与输出，仅运行电机阶跃调试任务。
2. 默认在 `20%~70%` 之间每 5 秒改变 5 个百分点，避免低占空比死区和满占空比饱和。
3. 记录稳态占空比、稳态转速、上升时间和超调量。
4. 首先设置 `Ki=0、Kd=0`，逐步增加 `Kp`。
5. 再增加较小的 `Ki` 消除稳态误差，最后按需要加入少量 `Kd`。
6. PID 控制周期建议保持在 10~20 ms，并使用实际采样周期调用 `speed_pid_update()`。

默认不提供虚构的 `Kp/Ki/Kd` 数值，因为参数与电池电压、负载、轮径和机械阻力直接相关，应由实机阶跃数据确定。
